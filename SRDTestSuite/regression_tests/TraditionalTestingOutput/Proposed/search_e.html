Search in source code engine

  • File contains:
    searches for test cases having words in the source code matching any of the given words. Matching is not case sensitive.
    • Exact:
      This option matches whole words. For instance "alloc" will not match "malloc" or "realloc".
    • Contains:
      This option matches substrings in words. For instance, "new" will match "MyNewVector" or "newbie" or "aNewStuff". (This is MUCH slower than Exact.)

  • File name:
    searches for test cases with file names matching any of the given words.
    • Regex:
      This option matches regular expressions to file names. (This is MUCH slower than otherwise.)
